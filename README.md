# Python Project 1 - Plant care tracker
An application for plant enthusiasts to track their houseplants' care schedule and maintenance history using a Python application.

This repo includes 3 .py  files that contain differnt functions to run the following application:

1. Add a new plant to the collection
2. Record a plant care activity
3. View plants due for care
4. Search plants by name or location
5. View all plants

app.py file that contains the code for the application streamlit.

2 csv files that will be used to record data
